Profile Editing

HTML EDITORS AND BEAUTIFCATION

  • Tess' HTML Editor -- This is a live editing platform from Tess.
  • HTML Edit - Simpler layout than the Tess' HTML Editor and also live.
  • Scratchpad - This has a more sophisticated design but also comes with quite its flaws. For one, links won't work in this website. It will just fuck it up.
  • Code Beautifier - Beautify your code so it's easier to read.
  • Code Compressor - On the other hand, you can also follow this link to compress your code and make it harder to read.

SCREEN CHECK AND DESIGN

  • Screenfly - You can check how your code looks in different resolutions.
  • ShareX - Want to take a picture, video, or gif of your screen fast and easy? This is the best one I found so far.
  • UI Gradients - If you're looking for aesthetic and complementary color gradients. This is the place.
  • Design Seeds - If you're looking for a good color palette, you might be able to find one here.
  • Color Hunt - Randomly spits out color palettes until you find one that you might like.
  • Transparent Textures - I usually find my background textures here. They're transparent and are seamless so they're virtually perfect for any use!
  • Canva Palette Generator - Grab the color palatte from an image! I use this a lot.
  • Canva Color Palettes - Type in a keyword or theme and Canva will show you an array of color palettes that fit that style.
